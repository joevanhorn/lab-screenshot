# Lab: Identity Security Configuration

## Step 1: Launch the Admin Console

1. On the lab guide page, find the **Launch** button
2. Click **Launch** to open the Admin Console in a new tab

[SCREENSHOT: Admin Console dashboard showing user count and application stats]

## Step 2: View Users

1. In the Admin Console navigation bar, click **Users**
2. Review the list of users showing their names, emails, and status

[SCREENSHOT: Users list page showing the table of users with status indicators]

## Step 3: Configure Settings

1. Click **Settings** in the navigation bar
2. Type **TaskVantage** into the Organization Name field
3. Type **30** into the Session Timeout field
4. Click **Save Changes**

[SCREENSHOT: Settings page showing the filled form with success message visible]

## Step 4: Return to Dashboard

1. Click **Dashboard** in the navigation bar
2. Verify the dashboard loads with the stats cards

[SCREENSHOT: Dashboard page showing the stats cards and recent activity table]
